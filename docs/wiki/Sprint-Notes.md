# Sprint Notes

This page serves as a directory and template for our Agile sprint notes and retrospectives. The team uses an Agile methodology emphasizing iterative development cycles to build the Suburban-SOC pipeline.

## Sprint Template

To document a new sprint, copy this template:

```markdown
### Sprint [Number]: [Date Range]

**Goal:** [Brief description of the main objective for this sprint]

#### Completed Milestones & Tasks:
- [x] Task 1
- [x] Task 2

#### Retrospective:
* **What went well:** ...
* **What could be improved:** ...

#### Action Items for Next Sprint:
- [ ] Action item 1
```
